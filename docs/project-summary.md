# Project Summary

## Project Purpose

AI Business Automation Hub with n8n is a portfolio project that demonstrates how to orchestrate lead operations from intake through qualification, routing, CRM preparation, and internal notification.

## Key Capabilities

- Webhook-based lead intake and validation
- Lead normalization and AI-style qualification simulation
- Priority-based routing and follow-up recommendations
- CRM-ready data preparation with basic failure handling
- Internal notification message generation
- Reusable helper API for shared transformation logic

## Stack Summary

- n8n workflows for orchestration
- FastAPI helper service for reusable logic
- Docker Compose for local development
- JSON sample payloads for demonstration and documentation

## Workflow Summary

- Lead Intake Workflow: receives and validates raw inbound lead payloads.
- AI Qualification & Routing Workflow: scores and categorizes leads, then routes by priority.
- CRM Sync & Notifications Workflow: prepares CRM sync records, audit logs, and internal alerts.

## Portfolio Positioning

This repository is designed as a realistic, simulation-based business automation showcase suitable for GitHub portfolios, client discussions, and Upwork proposal references.
